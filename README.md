# How to Play???
-->There are total 4 options for every question
-->select the option from 1-4 of your choice 
-->press 0 and enter to skip the question
-->result will show the 
   +Total Questions
   +Correct Questions
   +Incorrect Questions
   +Skipped Questions

Have fun playing!
